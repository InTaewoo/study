## 2. 컴퓨팅

### 2.1 AWS EC2(Elastic Computing Cloud)  *중요*
- elastic 탄력적 -> 컴퓨팅 서버를 애플리케이션 수요에 따라 늘리거나, 줄이거나가 가능하다.
- compute 컴퓨팅, 서버를 의미함
- cloud 구름
- 구름위에 탄력적인(줄이거나 늘리거나 가능한)서버들 -> 아마존 EC2인스턴스들
